# About

## Griddy SDK

Griddy is an open-source SDK for accessing NFL data from multiple API endpoints. It provides a unified, type-safe interface for Python and TypeScript developers.

## Features

- **Unified API**: Single interface for NFL.com APIs
- **Type Safety**: Full type hints (Python) and TypeScript definitions
- **Async Support**: Synchronous and asynchronous methods
- **Lazy Loading**: Sub-SDKs load on demand
- **Comprehensive Data**: Games, stats, rosters, Next Gen Stats, and more
