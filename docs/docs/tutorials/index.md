# Tutorials

Step-by-step tutorials to help you build applications with the Griddy SDK.

## Available Tutorials

### [First API Call](first-api-call.md)
Get started with your first API call to fetch NFL game data. Perfect for beginners.

**What you'll learn:**
- Installing the SDK
- Setting up authentication
- Making your first request
- Handling responses

### [Game Stats Dashboard](game-stats-dashboard.md)
Build a simple dashboard displaying game statistics.

**What you'll learn:**
- Fetching game data
- Extracting statistics
- Formatting output
- Building a basic web interface

### [Live Score Tracker](live-score-tracker.md)
Create a real-time score tracker for NFL games.

**What you'll learn:**
- Polling for live data
- Detecting score changes
- Building update notifications
- Rate limiting

### [Player Comparison](player-comparison.md)
Build a tool to compare player statistics.

**What you'll learn:**
- Fetching player stats
- Data normalization
- Visualization techniques
- Statistical analysis

## Prerequisites

Before starting these tutorials, ensure you have:

- Python 3.14+ or Node.js 18+
- Griddy SDK installed
- An NFL authentication token

## Quick Setup

=== "Python"

    ```bash
    pip install griddy
    ```

=== "TypeScript"

    ```bash
    npm install griddy-sdk
    ```
